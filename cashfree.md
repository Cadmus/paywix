# Cashfree Payment Gateway Integration

Please read the `Readme.md` for installation,

Setup
-----
```
CASHFREE_APPID = "*****************"
CASHFREE_SECRETKEY = "****"
CASHFREE_MODE = "TEST|LIVE"
CASHFREE_SUCCESS_URL = "/success url"
CASHFREE_FAILURE_URL = "/failure url"
CASHFREE_PAYMENT_BY_URL = True | False
```
